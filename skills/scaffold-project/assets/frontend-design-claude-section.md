## Frontend design system

This stack uses **React + Vite + TypeScript + Tailwind** with **shadcn/ui-style** components. Read `docs/frontend-design-system.md` before planning or implementing **marketing/landing** UI (Bundui **Cosmic**) or **authenticated dashboard** UI (Bundui **Dashboard** kit). Differentiate each product through **branding** and **structure** (nav, density, information architecture) while staying inside that component vocabulary. If `mockups/` or the PRD conflicts with a kit default, follow the PRD and mockups.
